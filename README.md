# trend_vwap_chikou_daytrade_1h.pine

TradingView Pine Script indicator for a 1-hour day-trading workflow.

## Main Conditions

- TrendSignal direction
- 25 EMA side
- Ichimoku Chikou span side
- Weekly pivot side
- Signal appears when at least 3 of 4 conditions align

## Notes

- Designed for TradingView Pine Script v6.
- This repository contains the latest working version created in Codex.
